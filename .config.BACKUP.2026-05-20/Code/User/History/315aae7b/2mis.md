# Idea de prollecto

## Planteamiento

Editor moderno AI first, con navegador y previsualizacion de ventanas con hot reload

1. Crear un editor con funcionalidades modernas que se conecte a un vps donde esstara la instancia de una distro linux real donde ocurre el desarrollo
2. El editor deve ser autonomo con ia y permir las capacidades avansadas de editores modernos
3. El editor al estar en un vps dedicado es capas de mantener la ejecucion de tareas aunque el usuario no este ŕesente
4. Se conectara a el vps via ssh y debe cumplir con los estandares modernos de editores/agentes tipo claude code o openclaw
5. git integrado, modular y extencible

## Consideraciones

1. multiplataforma
2. ligero
3. simple e intuitivo
