# Cambios en el stilo

## Rediseño de la interfas

ventajas:
1- mas simple
2- eficientemente revisable

Pautas de el diseño
1- una sola seccion
2- proyectos con mas espacio y visibilidad
3- correciones automaticas para pantalles reducidas

## Explicacion

Punto #1:
En lugar de usar 2 secciones (hero, proyects) ahora mostraremos todo el contenido en un solo lugar aprobechando toda la pantalla
Punto #2:
la seccion unica esta dividida 40%|60%, hero|proyectos en ese orden
Punto #3:
los componentes se reodenaran en dispocicion horizontal o vertical segun el dispositivo

_horizontal_
hero | proyets

_vertical_
hero
/
proyets
