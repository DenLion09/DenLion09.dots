# Specs

## Objetivo

crear una pagina de inicio simple y moderna para un blog personal
que sea bisualmente atractiva, funcional y adaptable a multiples dispositivos

## Estructura propuesta

### Cabecera (header)

- menu de tres barras para saltar entre secciones extras
- nombre del blog (debe tener dos verciones una simple para pantallas pequeñas y una mas grande para pabtallas grandes)
- obciones de login / cuenta en caso de estar logiado

### Main

- todo el contenido del blog (cargado desde archivos)
- diseño responsive
- en el lateral deve conterner un contenedor reducido para sintecis de otras notas (este solo se debe ver en pantallas grandes)

### Pie de pagina (footer)

- informacion de contacto
- enlaces adicionales (redes sociales, recursos, productos, etc)

## Funcionalidades

- Diseño responsive
- modo admin para cargar las paginas del blog en desde archivos en formato _.md_ y las demas informaciones de contact
- integracion de login y usuarios:
- - recivir notificaciones de nuevos blogs
- - autocompletar campos de correo en la seccion de contacto

## Specs esteticos

- Diseño simple y moderno
- colores neutros con efectos simples y elegantes
- tipografia legible y consistente
