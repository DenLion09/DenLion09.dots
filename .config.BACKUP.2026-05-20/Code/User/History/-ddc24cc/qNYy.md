# Specs

## Objetivo

crear una pagina de inicio simple y moderna para un blog personal
que sea bisualmente atractiva, funcional y adaptable a multiples dispositivos

## Estructura propuesta

### Cabecera (header)

- menu de tres barras para saltar entre secciones extras
- nombre del blog (debe tener dos verciones una simple para pantallas pequeñas y una mas grande para pabtallas grandes)
- obciones de login / cuenta en caso de estar logiado

### Main

todo el contenido del blog
