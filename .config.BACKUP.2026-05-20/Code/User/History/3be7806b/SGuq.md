# Radar Tech

El **Radar Tech:** Es una destrateia que ayuda a visualizar y eevaluar tecnologias emergentes y consolidadas. Facilita la toma de deciciones informadas sobre que tecnologias adoptar, experimentar o mmantener bajo vigilancia. Imagina un radar que escanea el horizonte tecnologico, identificando las señales mas relevantes para tu proyecto.

Un radar tech ees una representacion graffica que categoriza tecnologias en funciones de su estado de adopcion y madurez. Generalmente, se divide en cuatro cuadrantes concentricos: A`doptar, `Trial`, `Evaluar`y`Hold`.
