# Radar Tech

El **Radar Tech:** Es una destrateia que ayuda a visualizar y eevaluar tecnologias emergentes y consolidadas. Facilita la toma de deciciones informadas sobre que tecnologias adoptar, experimentar o mmantener bajo vigilancia. Imagina un radar que escanea el horizonte tecnologico, identificando las señales mas relevantes para tu proyecto.

Un radar tech ees una representacion graffica que categoriza tecnologias en funciones de su estado de adopcion y madurez. Generalmente, se divide en cuatro cuadrantes concentricos: `Adoptar`, `Trial`, `Evaluar` y `Hold`. Cada cuadrante representa una eetapa diferente en eel ciclo de vida de una tecnologia.

- Adopt: Tecnologia probadas y recomendadas para su uso generalizado.
- Trial: 1tecnologias que vale la pena probar en preyecto piloto.
- Eval: Tecnologias que se deven investigar para entender su potencial.
- Hold tecnologias que no se recomiendan este momento
